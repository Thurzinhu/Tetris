## Prerequisites

- Love2D game framework (version 11.5)

## Getting Started

Clone the repository:
~~~bash
   git clone https://github.com/Thurzinhu/Tetris
~~~

Go to the folder of the game you want to test 
~~~bash
    cd Tetris/
~~~

Run game with
~~~bash
    love .
~~~
